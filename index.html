Эквалайзер 1.4.3

Эквалайзер - это забавное приложение для браузера, позволяющее легко и бесплатно создавать собственные музыкальные визуализаторы и делиться ими с использованием новейших функций HTML5, таких как Canvas, Web Audio API и MediaRecorder.

Нажмите F1 переключить интерфейс.

Предупреждение

Это приложение лучше работает с Firefox. Я рекомендую использовать его, потому что некоторые функции недоступны в других браузерах.

Пример: вы не сможете перетащить песню в некоторых браузерах. Тем не менее, потоковая передача из SoundCloud должна быть возможна.
